# 마크다운



## 제목/소제목 (Heading)

- `#`의 개수에 따라 `h1`~`h6`까지 표현 가능하다.



## 리스트

- 순서가 없는 리스트 : `-`(Hypen), `*`(asterisk)
- 순서가 있는 리스트 : `1.` 사용

- `tab`, `shift + tab` 기호로 순위를 조정한다.



## 코드 블록
